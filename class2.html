Flexbox

Display: flex

  • Flexbox is an incredibly powerful tool!
  • By using display: flex; we can use flexbox to position our items inside a flexible box.
  • Flexbox allows us to quickly manupulate layout and display items where we want them on the page
  • Flexbox can help us space out the items in our nav bar

Wait, didn't we give our nav bar display: fixed?

That's right! We will need to make sure to target a different element.

How to Start

    
    

      

        
Item One

        
Item Two

        
Item Three

      

    

    
  
    
      .container {
        display: flex;
      }

Make sure to target the parent class!

Spacing Properties

    
      .container {
        display: flex;
        justify-content: space-around;
      }
    
  
    
      .container {
        display: flex;
        justify-content: center;
      }
      .container div {
        padding: 0 15px;
      }

Perfectly Centered!

    .container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 200px;
    border: solid 1px black;
}

.container div {
    padding: 0 15px;
}

We can even get our items to stack!

    
      .container {
        justify-content: center;
        flex-direction: column;
        align-items: center;
      }

Maybe we want our items reversed?

    
      .container {
        justify-content: center;
        flex-direction: row-reverse;
        align-items: center;
      }

Hero Images

What is a Hero?

  • Large banner image, prominently placed on a web page, generally in the front and center
  • First visual a visitor encounters on the site and its purpose is to present an overview of the site's most important content
  • Often consists of image and text, can be static or dynamic
Carousel Hero
Video Hero

Hero: HTML

Our Hero section should look a little something like this:

<section id="hero">
  
  
IMA Zebra

  Africa
</section>

Hero: CSS

Now is where the fun really happens!

#hero {
  background: url("images/zebra-hero.jpg") no-repeat;
  color: #fafafa;
  text-align: center;
}

Hero: CSS

Let's give it a height and some padding too.

#hero {
  background: url("images/zebra.jpg") no-repeat;
  color: #fafafa;
  text-align: center;
  height: 350px;
  padding: 25px 0;
}

Remember our Box Model. Padding adds to the height & width of elements.

So the height of our hero will be 400px

Hero: Profile Image

Let's make the profile image a little smaller.

We'll use CSS Targeting with the descendant selector to style the image.

  width: 150px;
}

Border Radius

Turning squares into circles with magic!

Okay, it's not really magic, it's just a bit of CSS3.

Border-Radius

Simply put, allows you to create rounded corners on boxes.

Designers rejoice!

Border-radius

20px radius on all corners

#hero img {
    border-radius: 20px;
}

Border-radius

10px radius on top left & bottom right

40px on top right & bottom left

#hero img {
    border-radius: 10px 40px;
}

Border-radius

10px radius on top left

20px radius top right

40px radius bottom right

80px radius bottom left

#hero img {
    border-radius: 10px 20px 40px 80px;
}

Border-radius

50% radius on all corners

#hero img {
    border-radius: 50%;
}

Back to our Hero!

Background-size

Notice how the image is too large for the section? Let's fix that with a new property called background-size 

#hero {
    background: url("images/zebra.jpg") no-repeat;
    color: #fafafa;
    text-align: center;
    height: 350px;
    padding: 25px 0;
    background-size: cover;
}

cover scales the image to the largest size such that both its width and its height can fit inside the content area.

Some other background image properties include:

  • background-position (center, right, left, etc.)
  • background-repeat(repeat, no-repeat and more!)

3-column Content Area

Why 3 columns?

Because 2 columns is so 2013

Or, because it's a comfortable width for readability. And because 3 is a pleasing design construct.

3 Column: HTML

Our code in structure.html already looks like this:

<section id="main-content">
  <div class="container">
    <section class="column">
      <!-- Content goes here! -->
    </section>
    <!-- Repeat .column x3 -->
  </div>
</section>

3 Column: CSS

Now that we have our 3 columns, we want them to appear next to each other. We can do this with flexbox!

.container {
      display: flex;
      justify-content: space-between;
}

We used space-between, because it gives us perfectly spaced columns without doing math! Don't forget padding (or margin) to give the columns some space.
