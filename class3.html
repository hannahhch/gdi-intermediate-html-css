Scalable Vector Graphics(SVG)

How is an SVG different from a JPEG or PNG image?

SVGs are scalable, meaning when you change their dimensions, they will still look smooth!

This is because unlike a reglular image, SVGs are made up of HTML

Example from W3schools

  • SVGs are ideal for creating icons or graphics that may change their size (ie: desktop to mobile)
  • They can be animated with CSS and JS
  • Are supported by all browsers (IE9 and up)

How to use SVG today

Use Adobe Illustrator, Sketch or another vector program, to create a high quality image.

Save it as a .svg file.

Include SVG

Use SVG as an image:

<img src="image.svg" alt="my-image">

Use SVG as a background image:

HTML:

<a href="/" class="logo">

}

FontAwesome

FontAwesome provides free SVG icons

Including them in our code is a little different than regular SVGs

First we will need a CDN (content delivery network) code from FontAwesome. This will go in the head of our HTML document.

The CDN codes tells our HTML file to include FontAwesome in our project!

Now we can browse the FA library for icons

FontAwesome Icons

Social Links: HTML

<section id="social">
  <ul class="social-icons">
    <li>
      <a href="..">
        <img src="images/facebook.svg" alt="Facebook" />
      </a>
    </li>
    <li>
      <a href="..">
        <i class="fab fa-linkedin-in"> />
      </a>
    </li>
<!-- Repeat for all social links you want to include -->
  </ul>
</section>

We're using a list for these links for the exact same reason we used them in the navigation. They are a list of links, and should be marked up accordingly.

Google Fonts

Google Fonts

The world of HTML has progressed beyond Times New Roman and Arial

Yay!

How do we use new and cool fonts?

Meet Google Web Fonts

Google has hundreds of free, open-source fonts that have been optimized for the web, and ready for us to use!

The service runs on Google's servers which are fast, reliable and tested. Google provides this service free of charge.

https://fonts.google.com/

In our example, we've used Lato for the body and Bree Serif for the headlines

You can use any font you'd like!

Using Google Fonts in 3 steps!

  1. Search the hundreds of font families, then add them to your collection.
  2. Compare and refine the collection - think about what styles you NEED.
  3. Grab the code that Google prepares for you put it in the head of your HTML doc!

Integrating into the CSS


body {
  font-family: 'Lato', sans-serif;
}

h1, h2, h3, h4, h5, h6 {
  font-family: 'Bree Serif', serif;
}

rgba colors & opacity

In CSS, we can choose colors with hexadecimal #0000FF or rgb(0, 0, 255)

In CSS3, there are ways to control opacity of a color too!

  • opacity - controls the opacity of an element

Gradients
